Little About Core Framework 4.0

If you see the scrollbars on the right hand side of different browsers it is the effect of core framework, but it is only supported by browsers that have prefix-webkit-vendors. This scrollbar will not be visible on tablet size (992px) and smartphone (600px) as both use touch to scroll content, so there is no need for scrollbars. If you are on the size of more than 992px then the scrollbar will be visible, this is intentionally made because the smartphone tablet does not require scroll. If you do not believe the Top Developer Core Framework has tested it on a computer chrome browser you can try on inpact elements with responsive mode.
